# 三角洲游戏助手 AI生成专用Prompt文档

# 文档基础信息

- **工具定位**：三角洲游戏助手（小麻虾电竞出品），开源、绿色、无毒、免登录、Windows端专属、低内存占用

- **开发语言**：Python

- **文档用途**：本地存储投喂AI专用，一键生成符合所有需求的完整Python代码，适配各类大模型与本地AI

---

# 一、核心功能明细（AI需严格实现）

## 1. 屏幕中心准星设置模块（辅助镭射瞄准）

- 默认显示在屏幕正中心，支持快捷键唤出/召回，默认快捷键F9

- 支持自定义调节：准星颜色、准星大小、外发光效果，参数可实时调整生效

- 窗口属性：强制置顶（游戏全屏/无边框均不遮挡）、透明背景、无边框、穿透鼠标输入（不影响游戏操作）、无任务栏图标

## 2. AI Skill 技能库模块

- **核心定位**：依托AI Skill能力封装的实用游戏技能库，属于RGA检索增强体系的前置技能分支，无需启用完整AI问答模块，即可轻量化调用专属游戏技能，贴合低占用需求。

- **细分AI Skill技能**：包含战备配置Skill（跑刀流、以小搏大流派，内置修脚枪专属配装与打法攻略）、撤离点攻坚Skill（露娜、红狼、乌鲁鲁三大角色专属炸撤离点点位、实操技巧），均为固化AI技能，调用无延迟。

- **界面形式**：独立Tab页分类展示，按Skill类型分区排布，内容直白易懂、轻量化无冗余，无需依赖Chroma数据库，默认加载不占用额外内存，适配工具绿色轻量化定位。

- 战备分类：跑刀、以小搏大（含修脚枪专属攻略）

- 炸撤离点点位：露娜、红狼、乌鲁鲁三大角色专属点位明细

- 界面形式：Tab页分类展示，内容清晰易读，轻量化展示无冗余

## 3. RGA检索增强AI问答（Chat模块，选配式）

- **核心定义**：搭载RGA检索生成增强能力，主打本地向量检索+智能问答，摒弃本地部署大模型，杜绝高额内存占用，贴合工具轻量化核心定位。

- **检索范围**：定向检索本地Chroma向量数据库，涵盖三角洲游戏配置、版本更新、实时物品价格、AI Skill战备攻略、AI Skill炸撤离点技巧等全品类资料，实现精准速答、贴合实战需求，拒绝无效泛化回答。

- **模块选配规则**：因内置Chroma、RGA相关依赖会使应用体积超50MB，故设为**用户选配模块**，默认关闭不加载、不占用任何内存与存储空间；用户手动开启配置后，再加载对应依赖与功能，兼顾功能完整性与工具轻量化，未配置时隐藏该功能入口。

- **界面与性能**：极简界面设计，配备提问输入框、发送按钮、问答历史展示区；仅在用户发起提问时触发检索+生成逻辑，无请求时静默运行，无常驻网络连接、无后台资源消耗，不拖慢工具运行速度。

### 选配模块配置步骤（AI需代码落地）

1. **功能启用入口**：主程序设置页新增「AI问答模块启用」开关，默认关闭；用户勾选开启后，提示重启工具生效，未开启时Chat Tab页隐藏/置灰不可用。

2. **依赖检测与安装**：工具启动时检测选配依赖（chromadb、sentence-transformers等），未安装则弹出提示，支持一键自动安装依赖，无需用户手动pip操作；也可提供手动安装指令，适配无外网环境。

3. **本地数据库初始化**：依赖安装完成后，首次启用自动初始化Chroma向量库，内置默认游戏资料+AI Skill专项数据集；支持用户手动更新数据库，导入最新物品价格、战备Skill、撤离点Skill资料，支持增量更新不重复录入。

4. **功能使用流程**：模块启用后，Chat页正常展示，用户输入提问→工具触发RGA逻辑→本地Chroma检索匹配资料→结合检索内容生成精准回答→展示问答结果；无匹配资料时返回兜底提示，不报错卡顿。

5. **模块关闭与降级**：用户关闭开关后，工具自动卸载/屏蔽选配依赖，释放内存与存储空间，回退至纯本地轻量化模式，不残留冗余文件，不影响核心功能使用。

## 4. 一键断网上传设置模块（核心功能）

- 仅断开网络上传，不断开网络下行，不影响游戏下载、语音、直播等下行功能

- 快捷键绑定，默认快捷键F10，一键切换断传/恢复状态

- 操作后触发顶层悬浮提示，提示强制置顶、不抢焦点、自动消失

- 提示位置支持切换：默认左上角，可调节至右上角、屏幕底部

## 5. 推广模块

每个功能Tab页底部设置推广栏，位置显眼且不影响核心功能操作，内容包含小麻虾电竞Logo、品牌Slogan、小程序二维码，采用静态资源加载，无额外内存占用，适配整体UI风格。

# 二、技术实现要求（AI需严格遵循）

## 1. 技术栈选型

- UI框架：PyQt6（轻量化配置，禁用冗余动画，降低内存占用）

- 快捷键监听：keyboard库（轻量无冗余，内存占用极低）

- 窗口置顶：Windows原生API（SetWindowPos）+ PyQt6置顶属性，实现游戏级强制置顶

- 准星绘制：PyQt6 QPainter（抗锯齿、支持发光效果绘制）

- 网络控制：Windows防火墙API+subprocess（系统级操作，稳定阻断上传）

- 基础数据存储：本地JSON文件（轻量化存储战备、点位数据，无数据库依赖）

- 选配AI问答：Chroma向量数据库（本地检索）、RGA检索生成逻辑、轻量语义向量化模型，仅用户选配后加载对应依赖，默认不打包进主程序，减少体积占用。

## 2. 性能硬性要求

- 默认模式（关闭AI问答选配模块）：运行内存占用≤50MB，应用体积远低于50MB，闲置状态内存占用进一步降低，无内存泄漏、无后台缓存堆积

- 选配模式（开启AI问答模块）：允许内存、体积小幅上浮，需做好资源管控，闲置时自动释放检索缓存，关闭模块后彻底释放资源，仍保证不抢占游戏系统资源

- CPU占用极低，全程无高负载运行，不影响游戏流畅度，无卡顿闪退情况

- 免安装、免登录，纯绿色单文件运行，无后台常驻进程，选配模块支持一键关闭、无残留卸载。

## 3. 代码规范要求

- 代码结构清晰，模块化拆分（核心功能模块、选配AI模块、UI模块、工具类模块），注释完整，易读易修改、易二次开发

- 无冗余依赖、无恶意代码、无广告捆绑、无后门程序，符合开源绿色标准

- 悬浮窗口（准星、断网提示）始终置顶，不闪烁、不闪退、不遮挡游戏核心画面，穿透鼠标操作不影响游戏操控

- 选配模块逻辑严谨，依赖加载/卸载、数据库初始化/更新流程无bug，报错有兜底提示，不影响主程序运行。
